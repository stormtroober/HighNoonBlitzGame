class NetworkConsistencyChecker(callback: ConsistencyCheckCallback, debugMode: Boolean = false)

Checks the consistency of client connections in the network.

This checker verifies that all clients are connected to each other and calls the provided callback with the result. It supports an optional debug mode for logging.
